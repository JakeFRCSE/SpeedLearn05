# 스피드런 5기 스터디

## 1. 스터디 규칙

- 매주 책 읽고 스터디 시간에 토론

- 스터디 시간: 매주 목요일 21시 ~ 23시, 일요일 18시 ~  20시

- 스터디 장소: 새벽벌 도서관 또는 창의마루

- 교재: 밑바닥부터 시작하는 딥러닝 1, 2권
